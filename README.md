## miniE-commerce
A mini e-cormerce application react, react hook and redux

## Features
- View products
- Add to Cart
- Remove from cart
- Checkout with Pay Stack

See front end at [Shop Now](https://mini-ecommerce.herokuapp.com/)

### How to use
- Clone repository
_ Run ```yarn build```
- Run ```yarn start```
- To run in development, run ```cd ecommerce-ui && yarn start```
